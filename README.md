## ✨ Features

### 🎯 Core Functionality
- **Appointment Booking** — Select doctor, specialty, date, time slot with full form validation
- **Rescheduling & Cancellation** — Seamless appointment management with confirmation flows
- **Queue Management** — Real-time waiting room tracking with check-in/check-out
- **Token Generation** — Automated queue numbering system
- **Walk-in Patient Handling** — Quick registration for unscheduled visits

### 👥 Role-Based Interfaces
| Role | Capabilities |
|------|-------------|
| **Administrator** | Full system access, analytics, clinic settings, holidays |
| **Receptionist** | Patient registration, booking, queue, check-in/out |
| **Doctor** | Daily schedule, patient lists, consultation status |
| **Nurse** | Patient vitals, queue assistance, notifications |

### 📊 Dashboard & Analytics
- Today's appointments, completed visits, pending confirmations
- Interactive charts (weekly/monthly appointment overview)
- Doctor availability indicators with real-time status
- System-wide analytics with trend visualization

### 📅 Calendar System
- **Daily View** — Hour-by-hour appointment timeline
- **Weekly View** — Full week grid with appointment slots
- **Monthly View** — Calendar with event indicators and click-to-view

### 🔍 Advanced Search & Filtering
- Global search across patients, doctors, and appointments
- Filter by patient name, doctor, department, date, status, type
- Multi-column sorting (ascending/descending)
- Real-time search results dropdown

### 🌗 Theme Support
- Light Mode & Dark Mode with localStorage persistence
- Smooth transitions between themes
- Consistent design across both modes

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Muhammad9985/portfolio.git

# Navigate to project
cd portfolio/apmt

# Open in browser (using XAMPP, Live Server, or directly)
# Option 1: Place in htdocs and visit http://localhost/apmt
# Option 2: Open index.html directly in any modern browser
# Option 3: Use VS Code Live Server extension
```

> No build tools, no dependencies to install. Just open and go.
